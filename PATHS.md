# Chess-Mate: Backend & Frontend Paths

Use these in the terminal (e.g. `cd` or in scripts).

## Absolute paths

| Role     | Path |
|----------|------|
| **Project root** | `~/chess-mate` or `/Users/davidreichardt/chess-mate` |
| **Backend**      | `~/chess-mate/backend` or `/Users/davidreichardt/chess-mate/backend` |
| **Frontend**     | `~/chess-mate/frontend` or `/Users/davidreichardt/chess-mate/frontend` |

## From project root

```bash
cd chess-mate/backend    # or: cd backend (if you're already in chess-mate)
cd chess-mate/frontend   # or: cd frontend (if you're already in chess-mate)
```

## One-line copy-paste (from home directory)

```bash
# Backend
cd /Users/davidreichardt/chess-mate/backend

# Frontend
cd /Users/davidreichardt/chess-mate/frontend
```

## Common commands (run from the right directory)

- **Backend:** `npm run dev` · `npm run db:seed` · `npm run db:validate-lessons`
- **Frontend:** `npm run dev` · `npm install`
